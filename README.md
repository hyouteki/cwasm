## C Wasm Js Template
Trying to get my ass to do some web dev, but can't get over with C. <br>
Idea by [zozlib.js - tsoding](https://github.com/tsoding/zozlib.js)

## Getting started
Install the following dependencies
``` bash
sudo apt install clang lld-14 wabt
```
``` bash
chmod +x build.sh
./build.sh
```
